Install
------------------------------------------------------------------------------

``adam_awesome`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install adam-awesome

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade adam-awesome
